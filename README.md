# Post-Disaster Road Accessibility Assessment

This repository contains the implementation of a post-disaster road accessibility assessment framework based on SegFormer-B2, graph analysis, and OpenStreetMap (OSM).

## Features

- Road segmentation using SegFormer-B2
- U-Net and ENet model comparison
- Skeleton extraction
- Graph generation
- OSM graph comparison
- Road accessibility analysis
- Test-Time Augmentation (TTA)

## Project Status

🚧 This repository is currently under active development.

The complete documentation and pretrained models will be added after the publication of the associated research paper.
